## LineageOS issues on Sony SM8450 (Nagara) devices

This repository hosts the issues.

## Supported devices:
* **Xperia 1 IV**:
	* XQ-CT72 (Asian variant)
	* XQ-CT54 (European variant)
	* XQ-CT44 (SIM-lock free Japanese variant)

* **Xperia 5 IV**:
	* XQ-CQ72 (Asian variant)
	* XQ-CQ54 (European variant)
	* XQ-CQ44 (SIM-lock free Japanese variant)

For other models of these phones, please contact the developers by opening an issue. Don't ask for ETA!

### Working hardware
- Display (this includes DT2W)
- LED (only when charging)
- Audio
- Bluetooth
- Calls (earpiece, speaker, wired and Bluetooth headphones)
- GPS
- WLAN (connect and hotspot)
- RIL
- VoLTE
- Camera
- Wireless charging
- Fingerprint
- Sensors
- Vibrator
- SD card

### Not working
- LED doesn't blink on notification
- VoWiFi
- Switching from 1080p to 4K cause everything on the screen being tiny (probably won't be fixed)
